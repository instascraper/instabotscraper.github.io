What is it?

Instagram Scraper with repost and face recognition is a Python module, which has implemented the Instabot Instagram API, Instagram Scraper is also part of the instagrambot/instabot organisation these are some of the instagram scraper functions:

  • Only follow real humans with machine learning,
  • Scrape images of real humans,
  • Repost scraped images.

  • More scripts

Features

  • Face Detection AI

    Instagram scraper is smart enough: it recognise real humans which more likely going to subscribe to your profile. That will help you attract real people.

  • Repost Real Images

    Instagram scraper is a bot which repost images found with face detection. Instagram Scraper does not repost the same image twice

  • Fully open-source

    All codes are hosted on GitHub and has been checked by thousands of developers. No one can steal your password via Instabot.
